Each Module should have a header that contains\+:
\begin{DoxyEnumerate}
\item The name of the module.
\item What the module does.
\item Module functions overview.
\item The date of creation.
\item The author of the module. 
\end{DoxyEnumerate}\hypertarget{md_00-project_autotoc_md2}{}\doxysection{\texorpdfstring{Naming Conventions}{Naming Conventions}}\label{md_00-project_autotoc_md2}

\begin{DoxyEnumerate}
\item Local Variables\+: Snake case or Flat case (i.\+e. {\ttfamily variable\+\_\+name}, {\ttfamily variablename})
\item Global Variables\+: Pascal snake case (i.\+e. {\ttfamily Global\+\_\+\+Variable})
\item Constants\+: Capital letters only (i.\+e. {\ttfamily CONSTANT})
\item Functions\+: Camel case (i.\+e. {\ttfamily function\+Name})
\item Classes\+: Pascal case (i.\+e. {\ttfamily Class\+Name})
\end{DoxyEnumerate}

{\itshape Avoid including digits as much as possible.} \hypertarget{md_00-project_autotoc_md3}{}\doxysection{\texorpdfstring{Code Format}{Code Format}}\label{md_00-project_autotoc_md3}

\item Braces\+: Open brace on the same line as the control statement.
